## Login / Registration

**Title:** User Registration  
As a user, I want to register with username, email, and password so that I can create an account.  

**Acceptance Criteria:**  
1. Users can enter valid details and click “Sign Up.”  
2. An error message is shown for invalid or missing input.  
3. User details are saved in local storage.  

**Story Points:** 3

---

**Title:** User Login  
As a user, I want to log in using my email and password so that I can access my account.  

**Acceptance Criteria:**  
1. Users can log in with correct credentials.  
2. An error message is displayed for incorrect credentials.  
3. Feedback is provided if fields are empty.  

**Story Points:** 3

## Favorites

**Title:** Add/Remove from Favorites  
As a user, I want to add or remove exercises from Favorites so that I can manage my saved content.  

**Acceptance Criteria:**  
1. Heart icon toggles between outlined and filled.  
2. Button text updates to “Remove from Favorites” when added.  
3. Users can add or remove items anytime.  

**Story Points:** 3

---

**Title:** My Favorites Screen  
As a user, I want a “My Favorites” screen so that I can view all saved items in one place.  

**Acceptance Criteria:**  
1. Displays a list of saved items with title, category, and duration.  
2. Users can tap items to view details or start the activity.  

**Story Points:** 2

---

## Daily Reminders

**Title:** View Calendar  
As a user, I want to view the current month’s calendar so that I can select dates for reminders.  

**Acceptance Criteria:**  
1. Display all days of the month.  
2. Navigation arrows to move between months.  

**Story Points:** 2

---

**Title:** Set Reminder Time  
As a user, I want to select a date and time for a reminder so that I can schedule it properly.  

**Acceptance Criteria:**  
1. Display default text: “Selected Date: None” and “Selected Time: HH:MM.”  
2. Allow selecting a specific time and date.  

**Story Points:** 2

---

**Title:** Add/Delete Reminder  
As a user, I want to add or delete a reminder so that I can manage my schedule.  

**Acceptance Criteria:**  
1. “Add Reminder” button schedules reminder.  
2. Red “Delete” button removes a reminder from the list.  

**Story Points:** 3
